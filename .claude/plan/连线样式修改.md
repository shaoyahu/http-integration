# 连线样式修改

## 任务需求
将工作流各个节点之间连线的样式，由实线修改为虚线。

## 实现方案
在 Canvas 绑定连接曲线的绘制函数 `drawCurveConnection` 中，使用 `ctx.setLineDash([5, 5])` 设置虚线样式。

## 开发过程

### 修改文件
- `src/components/workflow/types.ts`

### 具体修改
在 `drawCurveConnection` 函数（lines 281-289）中：
- 在 `ctx.beginPath()` 前添加 `ctx.setLineDash([5, 5])` 开启虚线
- 在 `ctx.stroke()` 后添加 `ctx.setLineDash([])` 恢复实线

```typescript
ctx.strokeStyle = color;
ctx.lineWidth = 2 / scale;
ctx.lineCap = 'round';
ctx.setLineDash([5, 5]);  // 新增：开启虚线
ctx.beginPath();
ctx.moveTo(start.x, start.y);
ctx.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, end.x, end.y);
ctx.stroke();
ctx.setLineDash([]);      // 新增：恢复实线
```
