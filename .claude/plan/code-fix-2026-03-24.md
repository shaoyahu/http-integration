# 代码修复计划 - 2026-03-24

## 高优先级修复

### 1. SSRF 漏洞修复
**文件**: `server/index.js`
**问题**: `/api/proxy` 没有 URL 验证，可访问内网服务
**修复方案**:
- 添加 `net.isIP()` 检查，阻止私有 IP 和 localhost
- 阻止 `localhost`, `127.0.0.1`, `0.0.0.0`
- 阻止私有 IP 段: 10.x.x.x, 172.16-31.x.x, 192.168.x.x

### 2. 代理请求超时
**文件**: `server/index.js`
**问题**: 代理请求无超时限制
**修复方案**:
- 添加 30 秒超时控制
- 使用 `req.setTimeout()` 或 AbortController

### 3. 默认密码加固
**文件**: `server/index.js`
**问题**: 新用户默认密码 `123456` 太弱
**修复方案**:
- 要求首次登录必须修改密码
- 增加密码强度要求提示

### 4. admin 后门移除
**文件**: `server/index.js` 多处 (lines 214, 308, 589)
**问题**: username "admin" 可绕过权限检查
**修复方案**:
- 移除所有 `username === 'admin'` 的特殊逻辑
- 统一使用 `USER_ROLE.ADMIN` 进行权限判断

### 5. 请求体大小限制
**文件**: `server/index.js`
**问题**: express.json() 无大小限制
**修复方案**:
- 添加 `express.json({ limit: '1mb' })`
- 添加 `express.urlencoded({ limit: '1mb', extended: true })`
