# 管理后台页面

管理后台是拥有管理后台权限的用户才能进入的页面。管理后台目前包括三个部分：
- 管理后台首页
- 用户管理页面
- 身份管理页面

## 技术栈

- **图表库**: recharts
- **UI 组件库**: Ant Design (antd)
- **权限控制**: 沿用现有 ProtectedRoute + isAdminUser() 体系

## 页面布局

```
+--------------------------------------------------+
|  Header (AppShell 自带)                          |
+--------+-----------------------------------------+
|        |                                         |

+--------+-----------------------------------------+
```

## 一、管理后台首页

管理后台首页展示开发平台中请求和工作流的数据量级信息，需要通过以下两种方式来展示数据：
1. 使用图表
2. 使用数据表

### 1.1 使用图表

使用图表时，满足以下几点要求：
1. 有请求和工作流的数量总和展示
2. 能直观看出请求和工作流在总数中的占比
3. 代表请求和工作流图表的颜色应该有明显对比（请求: #1890ff 蓝色，工作流: #52c41a 绿色）
4. 鼠标移动时会有 tooltip 展示具体数值
5. 图表类型: 饼图或环形图

### 1.2 使用数据表

使用数据表时，需要满足以下几点要求：
1. 数据表能正确展示请求和工作流的数量数据和占比
2. 表头使用深色背景 (#f0f2f5)，表本体白色背景
3. 数据行 hover 时有高亮效果

### 1.3 布局要求
- 图表和数据表左右并排展示
- 各占 50% 宽度
- 使用 Card 组件包裹

## 二、用户管理页面

### 2.1 功能列表
- [ ] 用户列表展示（分页、排序）
- [ ] 用户搜索（按用户名、邮箱）
- [ ] 创建用户
- [ ] 编辑用户（修改基本信息、分配角色）
- [ ] 删除用户（需二次确认）
- [ ] 禁用/启用用户

### 2.2 列表字段
| 字段 | 说明 |
|------|------|
| 用户名 | string |
| 邮箱 | string |
| 角色 | 关联身份管理中的身份 |
| 状态 | 正常 / 禁用 |
| 创建时间 | datetime |
| 最后登录 | datetime |

### 2.3 交互细节
- 删除按钮需要确认对话框
- 操作后显示成功/失败 toast
- 加载中显示 skeleton
- 空状态显示"暂无用户"

## 四、API 接口

### 4.1 现有接口（已实现）
```
GET  /api/admin/stats          # 获取统计数据
POST /api/auth/register        # 注册
POST /api/auth/login           # 登录
GET  /api/admin/users          # 获取用户列表
POST /api/admin/users          # 创建用户
PUT  /api/admin/users/:id      # 更新用户
DELETE /api/admin/users/:id    # 删除用户
GET  /api/admin/identities     # 获取身份列表
POST /api/admin/identities     # 创建身份
PUT  /api/admin/identities/:id # 更新身份
DELETE /api/admin/identities/:id # 删除身份
```

### 4.2 需要后端配合
- [ ] `GET /api/admin/stats` 返回请求数量和工作流数量

## 五、权限控制

### 5.1 权限层级
- **超级管理员**: 可访问所有管理后台功能
- **普通管理员**: 可访问管理后台首页和查看用户/身份
- **普通用户**: 不可访问管理后台

### 5.2 实现方式
- 使用现有 `isAdminUser()` 检查
- 新增 `AdminLayout` 组件统一布局

## 六、文件结构

```
src/
├── pages/
│   ├── AdminPage.tsx              # 首页
│   ├── UserManagementPage.tsx    # 用户管理
│   └── IdentityManagementPage.tsx # 身份管理
├── components/
│   └── admin/

│       ├── StatsChart.tsx         # 统计图表
│       └── StatsTable.tsx         # 统计表格
└── store/
    └── adminStore.ts              # 管理后台状态
```

**注意**：侧边栏导航已在 AdminLayout 中实现，不需要额外的路由配置。导航通过侧边栏按钮切换页面内容。
