# 工作流布局修改

## 任务需求
删除工作流页面中的搜索节点框框。

## 实现方案
移除 WorkflowToolbar 中的节点搜索功能，以及 WorkflowAddPanel 中的请求搜索功能。

## 开发过程

### 修改文件

#### 1. `src/pages/WorkflowPage.tsx`
- 移除 `addSearch` 和 `nodeSearch` state 变量
- 移除 `filteredAvailableRequests` 计算
- 移除 `setAddSearch('')` 调用
- 移除传递给 WorkflowToolbar 的 `nodeSearch` 和 `setNodeSearch` props
- 移除传递给 WorkflowAddPanel 的 `addSearch`、`setAddSearch`、`filteredAvailableRequests` props
- 改为传递 `availableRequests` prop

#### 2. `src/components/workflow/WorkflowToolbar.tsx`
- 移除 `nodeSearch` 和 `setNodeSearch` props
- 移除整个节点搜索输入框和下拉列表的 UI

#### 3. `src/components/workflow/WorkflowAddPanel.tsx`
- 移除 `addSearch`、`setAddSearch`、`filteredAvailableRequests` props
- 移除搜索输入框
- 保留请求列表展示，使用 `availableRequests` 替代 `filteredAvailableRequests`

- TypeScript 类型检查通过
