# 连线样式错误

## Bug 描述
工作流中各个节点之间的连线样式仍然是实线，需要修复为使用虚线连接。节点之间的已经存在的和后续会描绘的都要使用虚线。

## 需求关联
需求 #1 - 连线样式修改

## Bug 出现原因
`drawCurveConnection` 函数（位于 `src/components/workflow/types.ts`）在绘制贝塞尔曲线连接时，没有设置虚线样式。Canvas 2D API 默认使用实线。

## 解决方案
在 `drawCurveConnection` 函数中，使用 `ctx.setLineDash([5, 5])` 设置虚线样式。

## 开发过程

### 修改文件
- `src/components/workflow/types.ts`

### 具体修改
在 `drawCurveConnection` 函数（lines 281-290）中：
- 在 `ctx.beginPath()` 前添加 `ctx.setLineDash([5, 5])` 开启虚线
- 在 `ctx.stroke()` 后添加 `ctx.setLineDash([])` 恢复默认实线样式（避免影响后续绘制）

```typescript
ctx.strokeStyle = color;
ctx.lineWidth = 2 / scale;
ctx.lineCap = 'round';
ctx.setLineDash([5, 5]);  // 新增：设置虚线样式
ctx.beginPath();
ctx.moveTo(start.x, start.y);
ctx.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, end.x, end.y);
ctx.stroke();
ctx.setLineDash([]);       // 新增：恢复实线样式
```
